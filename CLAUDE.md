## Quick reminders

- Regenerate typed routes after any route change: `php artisan wayfinder:generate --with-form --no-interaction`.
- Match CI before handing work back: `composer ci:check` (lint + format + types + tests).
- Side effects (activity log, notification, mail) only via `event() → listener`.
- Don't hand-edit `resources/js/components/ui/*` (generated by shadcn).
- Shared/backend-reflecting TS types belong in `@/types`; page `Props` stay inline.
- Turkish (`tr`) is the primary locale; keep `lang/en` in sync.
- Unused-but-ready scaffolding (full shadcn set, permission hooks, enum helpers, alternative layouts, `media-gallery`) is deliberate — don't delete it as "dead code".
- Extend via the **module system** (Hook + Registry), not by hardcoding the sidebar: contribute menu/permissions/routes through the `{area}.menu.register` / `{area}.permissions.register` / `{area}.routes.register` hooks. See the [Module system](AGENTS.md#module-system-extensibility) section in AGENTS.md.
